# Amazon Sales Analysis Dashboard

## Project Overview

This project analyzes Amazon sales data using Power BI to uncover insights related to orders, revenue, product performance, fulfillment methods, and order status.

## Tools Used

* Power BI
* Power Query
* DAX
* SQL (Data Cleaning & Analysis)
* Excel

## Dataset Information

* Source: Kaggle
* Records: 128,976
* Period: April 2022 – June 2022

## Dashboard Pages

### Overview

* Total Orders
* Total Quantity Sold
* Total Revenue
* Cancellation Rate
* Revenue by Category
* Order Status Distribution

### Orders Analysis

* Orders by Month
* Orders by State
* Fulfillment Analysis
* Channel Analysis

### Sales Analysis

* Revenue Analysis
* Sales Trends
* Category Performance

### Product Analysis

* Product Category Insights
* Quantity Distribution
* Revenue Contribution

### Status Analysis

* In Transit Analysis
* Delivered Orders
* Cancelled Orders
* State-wise Status Distribution

## Key Insights

* Total Revenue: ₹78.59M
* Total Orders: 128.98K
* Cancellation Rate: 14.29%
* Set category generated the highest revenue.
* In Transit orders represented the largest order status segment.
