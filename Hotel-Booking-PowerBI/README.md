# Hotel Booking Analytics Dashboard | Power BI

## Project Overview

This Power BI project analyzes hotel booking data to uncover booking trends, customer behavior, revenue patterns, and cancellation insights.

The dashboard provides interactive visualizations and KPIs that help stakeholders monitor hotel performance and make data-driven decisions.

---

## Tools Used

* Power BI
* Power Query
* DAX
* Data Modeling

---

## Dataset

Hotel Booking Dataset containing:

* 17,707 hotel booking records
* Customer demographics
* Booking details
* Revenue information
* Cancellation status
* Distribution channels

---

## Dashboard Features

### KPI Cards

* Total Bookings: 17.7K
* Average Lead Time: 87 Days
* Cancellation Rate: 44.84%
* Average Daily Revenue (ADR): $95.99

### Visualizations

* Bookings by Country
* Monthly Booking Trends
* Revenue by Month
* Booking Status Distribution
* Meal Preference Analysis
* Special Booking Requests by Month

### Interactive Filters

* Country
* Customer Type
* Distribution Channel

---

## Key Insights

* Portugal (PRT) generated the highest number of bookings.
* August recorded the highest booking volume.
* Cancellation rate reached 44.84%.
* BB meal plan was the most preferred meal option.
* Average lead time was 87 days.
* Average Daily Revenue (ADR) was $95.99.
